# ls 

Menampilkan isi Folder.

Contoh:

$ls

Output:

Document
Downloads
Pictures

--

# pwd

Menampilkan lokasi direktori saat ini.

Contoh:

$pwd

Output:

~/Practice/folder1

--

# touch

Membuat file.

Contoh:

$touch file.txt

--

#mkdir

Membuat direktori baru.

Contoh:

$mkdir folder1.

--


#cp 

Membuat salinan file.

Contoh:

$cp file1.txt file1copy.txt

--

#rm

Menghapus file.

Contoh:

$rm file.xt

--

#chmod

Mengubah hak akses(permision) untuk suatu file.

Contoh:

$chmod 755 file.txt

Artinya:

-Angka pertama (7) - Pemilik (Owner): Membaca (4), Menulis (2), dan Eksekusi (1).
-Angka kedua (5) - Grup (Group): Membaca (4) dan Eksekusi (1).
-Angka ketiga (5) - Pengguna Lain (Others): Membaca (4) dan Eksekusi (1).

--

#mv

Memindahkan file ke direktori lain.

Contoh:

$mv file.txt ./folder2

--

#find

Mencari file.

Contoh:

$find -name "*.txt"
